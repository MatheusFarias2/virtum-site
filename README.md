# Virtum Site

Site institucional estático do ecossistema Virtum.

## Estrutura

- `index.html`: página inicial
- `style.css`: estilos globais
- `script.js`: interações
- `assets/`: imagens, ícones e arquivos de marca
- Pastas internas: `sobre`, `projetos`, `roadmap`, `documentacao`, `downloads`, `suporte` e `contato`

## Publicação

O site está preparado para GitHub Pages com domínio personalizado em `virtum.tec.br`.
