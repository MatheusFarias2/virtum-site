Como testar:
1. Extraia o ZIP.
2. Abra o arquivo index.html no navegador.
3. Compare a Home com as páginas Sobre, Projetos, Roadmap e Contato.
4. Confira se o topo está padronizado em todas.
5. Veja se a logo no header aparece com melhor contraste.
6. Confira se o botão do WhatsApp não fica cortado na lateral.
7. Se o favicon não aparecer de primeira, use Ctrl + F5.

Observação:
O site continua estático, sem back-end. O formulário ainda é demonstrativo e não envia mensagens reais. A publicação continua fora do escopo por enquanto.
